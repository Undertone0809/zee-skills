# editorial-room

Stress-test tweets, founder posts, launch announcements, product narratives,
and other public writing with an independent multi-agent editorial room.

The skill uses four conflicting reviewers, asks the primary agent to produce a
focused and a provocative direction, and sends those candidates to a fifth
blind reviewer before making the final editorial decision. It is designed to
improve clarity and credibility without polishing away the author's original
point of view.

## Install

```bash
npx skills add Undertone0809/zee-agent-skills/editorial-room
```

## Example prompts

```text
用 editorial-room 模拟一下这条 Founder Post 发出去后的效果。让不同视角互相冲突，最后给我一个明确版本，不要把我的争议观点删掉。
```

```text
Stress-test this launch tweet with an editorial room. I want one focused route,
one provocative route, and a final recommendation that preserves my voice.
```
